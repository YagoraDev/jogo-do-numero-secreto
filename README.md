# Jogo do Número Secreto

O Jogo do Número Secreto é um projeto front-end desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de lógica de programação, com foco em manipulação de DOM, eventos e interatividade com o usuário.

A aplicação consiste em um jogo simples onde o usuário deve adivinhar um número aleatório gerado pelo sistema entre 1 e 10.

## Tecnologias
- HTML5  
- CSS3  
- JavaScript (puro)

## Funcionalidades
- Geração aleatória de um número secreto entre 1 e 10 utilizando `Math.random()`
- Campo para o usuário inserir o palpite
- Dicas visuais indicando se o número secreto é **maior** ou **menor** que o palpite
- Contagem de tentativas até acertar o número
- Exibição da quantidade de tentativas ao acertar
- Botão "Novo jogo" para reiniciar a partida com um novo número aleatório
- Layout responsivo adaptado para desktop e dispositivos móveis

## Lógica de Programação

O projeto foi desenvolvido para testar e praticar conceitos como:
- Geração de números aleatórios com `Math.random()`
- Estruturas condicionais (`if/else`)
- Controle de fluxo do jogo
- Manipulação de eventos (cliques e inputs)
- Atualização dinâmica do DOM
